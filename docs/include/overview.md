## Install

1. install nodejs version > 8.0.0
2. install redis
3. cd koh_test
4. npm install

## Base URL
```
http://localhost:3000/api/1.0.0
```

## How to set env mode

```
- open terminal
- check current directory with pwd
- export NODE_ENV="<your env mode>" [production, development] | default is development
- echo $NODE_ENV
```
